HOW TO USE
================================================================================

1. SETUP (First Time)
   $ dotnet tool install microsoft.playwright.cli -g
   $ playwright install
   $ cd HeadlessBrowserAnalyzer
   $ dotnet build

2. RUN APPLICATION
   $ dotnet run

3. ENTER URL
   Enter a URL (or 'exit' to quit): example.com

4. VIEW RESULTS
   [Beautiful formatted analysis displayed]

5. SAVE OPTIONAL
   Save full HTML to file? (y/n): n

6. ANALYZE MORE OR EXIT
   Enter a URL (or 'exit' to quit): exit
